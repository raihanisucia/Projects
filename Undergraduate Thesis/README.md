# XGBoost and Random Forest Dose Prediction for Brain Cancer Radiotherapy

This repository contains the implementation of **XGBoost** and **Random Forest** machine learning models for predicting dose distribution metrics in brain cancer radiotherapy treatment planning.

The project was developed as part of my undergraduate thesis at Universitas Indonesia, focusing on predicting clinically relevant dosimetric parameters from radiomic features.

## Project Overview
The workflow includes:
- Data preprocessing
- Feature selection
- Model development using:
  - XGBoost
  - Random Forest
- Hyperparameter tuning
- Model evaluation
- Dose metric prediction workflow

## Objective
The objective of this project is to develop machine learning models capable of predicting clinically relevant dose distribution metrics for brain cancer radiotherapy planning.

The models utilize radiomic and dosiomic features extracted from planning CT images to estimate treatment planning dose metrics.

### Workflow
1. Data preprocessing
2. Feature selection
3. Train-test split
4. Model training
5. Hyperparameter optimization
6. Model evaluation
7. Dose metric prediction

## Dataset
The dataset used in this study is not included in this repository due to patient privacy, ethical considerations, and institutional confidentiality.

## Disclaimer
This repository is intended for research and educational purposes only. It should not be used for clinical decision-making without proper validation and regulatory approval.
