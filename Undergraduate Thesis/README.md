# XGBoost and Random Forest Dose Prediction

This directory contains the implementation of an XGBoost and Random Forest machine learning model for predicting dose distribution metrics in brain cancer radiotherapy treatment planning.

The project was developed as part of my undergraduate thesis at Universitas Indonesia and includes:
- Data preprocessing
- Feature selection
- XGBoost and Random Forest model development
- Model evaluation
- Prediction workflow

**Note:** The dataset is not included in this repository due to data privacy and confidentiality.
