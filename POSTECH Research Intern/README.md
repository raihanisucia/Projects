# KoalaDrive Piezomotor Step Size Analysis for Scanning Tunneling Microscopy

## Overview
This project presents the data analysis and performance characterization of **KoalaDrive**, a piezoelectric motor used in a Scanning Tunneling Microscope (STM). The analysis was conducted during my **research internship at the Scanning Tunneling Microscopy and Transport Laboratory (STM+T Lab), Pohang University of Science and Technology (POSTECH), South Korea**.

Using Python, the project processes experimental motion data, performs exploratory data analysis (EDA), and visualizes the relationship between operating parameters and motor displacement to evaluate the performance of the piezoelectric actuator.

## Objectives
- Analyze KoalaDrive displacement characteristics.
- Investigate the effect of voltage and frequency on step size.
- Compare motion behavior in different directions.
- Generate scientific visualizations for experimental interpretation.
