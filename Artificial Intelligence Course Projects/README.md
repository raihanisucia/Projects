# Pneumonia Detection using Mask R-CNN
This project implements a **Mask R-CNN** model to detect and localize pneumonia in chest X-ray images through instance segmentation. The model identifies lung opacity regions associated with pneumonia and visualizes the detected areas using bounding boxes.

This project was developed as part of a **Project-Based Learning (PBL)** assignment for the **Artificial Intelligence** course.

## Project Overview
The objective of this project is to explore the application of deep learning in medical image analysis by implementing Mask R-CNN for pneumonia detection on chest radiographs.

The project includes:

- Chest X-ray image preprocessing
- Dataset preparation
- Mask R-CNN implementation
- Pneumonia localization using bounding boxes
- Qualitative analysis of detection results

## Disclaimer
This repository was created for educational purposes as part of an undergraduate Artificial Intelligence course. It is intended to demonstrate the implementation of Mask R-CNN for pneumonia detection and should not be used for clinical diagnosis.
