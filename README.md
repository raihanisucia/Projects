## 🎓 Undergraduate Thesis

### Machine Learning-Based Radiotherapy Dose Prediction

Developed machine learning models to predict radiotherapy dose distribution for brain cancer treatment planning.

The project utilized radiomic and dosiomic features extracted from **178 clinical patient with brain cancer cases**, aiming to improve radiotherapy treatment planning efficiency through predictive analytics.

### Highlights

- End-to-end Machine Learning Pipeline
- Feature Engineering
- Hyperparameter Optimization
- Multi-output Regression
- Clinical Dose Prediction
- Performance Evaluation
